Introducción

El objetivo de este apartado es la determinación de una demanda potencial de contenedores, particularmente del tráfico interoceánico, que podrían ser captados por el proyecto Corredor Interoceánico de Guatemala

Las predicciones de demanda se calcularan hasta el año 2050 , el plazo aproximado para realizar la amortización de las inversiones necesarias para llevar a cabo el proyecto mencionado anteriormente.
